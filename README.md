# docker-golang-arm64
This is to build docker images for golang on arm64 systems.<br>
Therefore, you need a running arm64 machine to build images.<br>
<br>
Build example:<br>
```
docker build -t golang-arm64:1.6.3 -f Dockerfile.1.6.3 .
```
